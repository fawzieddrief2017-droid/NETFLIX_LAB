# 🍿 Mini AWS Netflix-Like Streaming Platform

Welcome to the **Mini Netflix Lab**! This project is an educational, small-scale video streaming platform built entirely on AWS. It demonstrates real-world streaming architecture patterns in a simplified, serverless, and Free-Tier-friendly manner.

## 🌟 Architecture Overview

The platform uses a fully serverless AWS architecture:
*   **Authentication**: Amazon Cognito User Pools securely manages user sign-ups and JWT tokens.
*   **API Layer**: Amazon API Gateway (HTTP API) protects endpoints natively using a Cognito Authorizer.
*   **Compute**: AWS Lambda functions (written in Python 3.12) handle business logic.
*   **Database**: Amazon DynamoDB (On-Demand) stores the content catalog (`Titles`) and user watch progress (`WatchHistory`).
*   **Storage & Delivery**: Amazon S3 stores pre-encoded MP4 videos, which are securely delivered to users via Amazon CloudFront using short-lived Signed URLs.
*   **Infrastructure as Code**: The entire stack is provisioned using HashiCorp Terraform.

## 📂 Repository Structure

*   **`infra/`**: Contains all the Terraform HCL files to provision the AWS resources.
*   **`src/`**: Contains the Python 3.12 source code for the Lambda backend functions (`catalog`, `stream`, `history`).
*   **`docs/`**: Contains the deep-dive documentation for the project:
    *   `spec.md`: The architectural feature specification.
    *   `data-model.md`: DynamoDB schema designs.
    *   `contracts.md`: API Gateway REST endpoint contracts.
    *   `quickstart.md`: Step-by-step deployment guide.

## 🚀 Getting Started

To deploy this project to your AWS account, you will need the AWS CLI, Terraform, and Python 3.12 installed on your machine.

**Please refer to the complete deployment guide located at [`docs/quickstart.md`](docs/quickstart.md) for step-by-step instructions!**

## 💡 Key Features
*   **Secure Video Delivery**: Users cannot access the S3 bucket directly. Videos are only streamable via temporary CloudFront Signed URLs generated by the backend.
*   **Resume Watching**: The platform automatically saves playback timestamps to DynamoDB, allowing users to pause a video and resume it later.
*   **Cost Protected**: The infrastructure relies entirely on serverless, pay-per-request components, making it ideal for a low-cost educational lab. It includes an AWS Budgets alarm configured for $5/month.
